#  Blog API Service

A simple **Nest.js REST API** for managing blog posts, built with **MongoDB Atlas** and deployable on **Render**.

---

##  Features
- Create, read, update, and delete blog posts (CRUD).
- MongoDB Atlas integration.
- RESTful endpoints.
- Ready for deployment on Render.

---

##  Installation
```bash
git clone https://github.com/mkbello/blog-api.git
cd blog-api
npm install
